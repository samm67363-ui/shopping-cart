### **Step 1: Create Razorpay Account** (2 min)
```
Go to: https://razorpay.com
Sign up with email
Verify your email
```

### **Step 2: Get TEST API Keys** (1 min)
```
Dashboard → Settings → API Keys
Copy: Key ID (starts with rzp_test_)
Copy: Key Secret (starts with rzp_test_)
```

### **Step 3: Update Configuration** (2 min)
```
File: c:\xampp\htdocs\html\razorpay_config.php
Update: RAZORPAY_KEY_ID with your Key ID
Update: RAZORPAY_KEY_SECRET with your Key Secret
Save: The file
```
